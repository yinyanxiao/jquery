#### node.js
+ 为什么要下载node.js？Node是JavaScript(前台编程语言，用来给HTML网页增加动态功能)的一个运行环境。只有有了node，才可以运行JavaScript。
+ node-v 检查版本号
#### JavaScript的组成
+ DOM (文档对象模型) 
+ BOM (浏览器对象模型) 
+ ECMPScript(js本质的东西,它规定了JavaScript的组成部分：语法、类型、语句、关键字、保留字、操作符、对象),主流是ES5。ES6也叫作ES2015.
